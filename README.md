# Landing Page Ademicon Consórcios - Alta Conversão 🚀

Uma landing page profissional, moderna e otimizada para a venda de consórcios da **Ademicon**. Este projeto foi desenvolvido com foco em **conversão de leads** via WhatsApp, utilizando as melhores práticas de design, SEO e performance.

---

## 📸 Visual do Projeto
O design é inspirado no site oficial da [Ademicon](https://www.ademicon.com.br/), utilizando sua paleta de cores característica (Vermelho Ademicon, Cinza e Branco) e uma estrutura focada em autoridade e clareza.

---

## ✨ Funcionalidades Principais

-   **Múltiplos CTAs (Call to Action):** Botões estratégicos em todas as seções para contato imediato via WhatsApp.
-   **Mensagens Personalizadas:** Cada botão de WhatsApp envia uma mensagem específica dependendo do interesse do cliente (Imóveis, Veículos, Serviços, etc.).
-   **FAQ Inteligente (Acordeão):** Seção de dúvidas frequentes com conteúdo oficial da Ademicon e animações suaves em JavaScript.
-   **Otimização SEO:** Meta tags completas para Google, Facebook (Open Graph) e Twitter.
-   **Design Responsivo:** Adaptado perfeitamente para dispositivos móveis (Mobile First) e desktop.
-   **Animações de Scroll:** Efeitos de revelação de elementos ao rolar a página para uma experiência de usuário premium.

---

## 🛠️ Tecnologias Utilizadas

-   **HTML5:** Estrutura semântica e acessível.
-   **CSS3:** Estilização moderna com Flexbox e Grid.
-   **JavaScript (Vanilla):** Lógica de interatividade e animações sem dependências externas.
-   **Font Awesome:** Ícones vetoriais de alta qualidade.
-   **Google Fonts:** Tipografia Montserrat para um visual corporativo.

---

## 🚀 Como Utilizar

1.  **Clonar ou Baixar:** Baixe os arquivos `index.html`, `style.css` e `script.js`.
2.  **Configurar WhatsApp:**
    *   Abra o arquivo `index.html`.
    *   Pressione `Ctrl + F` e procure por `5500000000000`.
    *   Substitua pelo seu número de WhatsApp real (ex: `5511999999999`).
3.  **Adicionar Imagens:**
    *   Para o banner principal, gere uma imagem usando as sugestões no arquivo `prompts_imagens.md`.
    *   Salve as imagens na pasta do projeto ou atualize os caminhos no HTML.
4.  **Hospedagem:** Basta fazer o upload dos arquivos para qualquer servidor estático (GitHub Pages, Vercel, Netlify ou Hostgator).
